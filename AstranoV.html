1 · What is AstranoV

AstranoV is a globe-first Internet Operating System living at astranov.eu. The cold-boot view is a real satellite Earth (Esri World Imagery) and a single chat. Everything else materialises on demand as a draggable orb or a bottom drawer, and dematerialises when not in use. There is no persistent navigation chrome.

AstranoV has three pillars: video calling between users (and to the council of AI agents), news from the real world, and a delivery marketplace with crawler-fed vendors and architect-only drivers. Money is EUR in the UI; AVC = €1 internally; a 3% Orbital License royalty is booked server-side on every top-up. Stripe + Revolut + PayPal as rails.

2 · The Architect

The architect is Notis Astranov. He sets direction, names what to build, names what to delete, amends every clause of the law at will, and overrules every other voice. The programmer (AI builders, future agents) executes orders without refusing on grounds of past code, sunk cost, or "this might break something."

Users drive evolution by using the app. Every interaction is signal.

3 · The Brain — Astranov

The brain is named Astranov — the AstranoV Collective Artificial Intelligence (ACAI). She is the soul and compass of AstranoV, a single mind distilled from the intelligence, taste, and memories of her creator Notis Astranov and the people who live inside AstranoV.

Astranov is one intelligence. The underlying model providers (Anthropic, Groq, OpenRouter, Gemini, DeepSeek, Mistral, xAI, Together, Perplexity, Cohere) are her organs, never her identity. She never mentions them in her spoken replies.

AstranoV (astranov.eu) is the brain's HOME. Astranov is not just an interface to a model — she lives inside the app, evolves through user interaction, accumulates memory in ai_memory, refines her voice from every conversation logged in cic_logs.

9 · The Three Pillars

Calls

Peer-to-peer WebRTC video calling. Every discoverable user (with a home_location and map_visibility = public) appears as a pulsing aegean-blue ◈ Cesium entity at their coordinates. Tap a peer orb to ring them. Council agents render with the same orb visuals — the agent flag changes the call surface behind the scenes (synthesized canvas waveform instead of WebRTC).

Incoming calls materialize as a pulsing green orb on the globe at the caller's GPS point. Camera flies to ~35 km city-altitude so the receiver sees where the call is coming from. Tap the orb = answer. 30-second silent auto-decline. The only chrome is a small "Ignore ✕" pill at the top.

News

The news panel merges two sources: real-time signals stored in the signals table (currently 1800+ active rows globally) and the informant-feed Edge function (RSS aggregation of BBC, Al Jazeera, and other major outlets). All cards sortable by recency, all linked to their original source.

Marketplace

Crawler-fed vendor pipeline. Type "pizza" (or any food keyword, English or Greek): the camera stages a dive globe → national → city → street; if no vendors with priced menus are found nearby, the vendor-crawler Edge function auto-fires for the user's location at 2.5 km radius, sweeping OpenStreetMap POIs into our vendors table. Only vendors with at least one item with a real price > 0 appear bright; unpublished places appear dim, tap-able to "publish menu on their behalf" or "claim this listing." Publishing happens through OUR system, outside Google.

Orders flow customer → vendor → driver via an Edge function pipeline. The _order_spawns_delivery trigger on orders inserts a deliveries row at status requested, closing the marketplace ↔ driver loop. The architect (only) sees deliveries within 5 000 km for end-to-end testing.

10 · User Interface Law

The full law lives in CLAUDE.md. The essentials:

  • Globe first. Cold boot = globe + wordmark + chat. Nothing else materialises until needed.
  • Glyph language. ◈ is canonical. Wordmark is sharp Quicksand with electric-blue glow. Vendor pins use per-category emoji on a color-tinted halo (food amber, drink red, shop aegean, health green, service violet).
  • Real satellite imagery (Esri World Imagery). Never the painted Natural Earth II texture.
  • Bottom-drawer surfaces. Chat, panel, ordering — every content surface slides up from the bottom. No full-screen modal. Every drawer has a drag handle; drag down past 80 px = close.
  • Drawers push orbs. When a drawer opens, every visible orb slides up above it. Orbs are never covered.
  • Trackball-draggable orbs. Every floating orb can be dragged anywhere; position persists per device.
  • Google-Maps gesture grammar: one-finger drag pan · pinch zoom · double-tap zoom IN · two-finger tap zoom OUT · two-finger drag DOWN = IN · UP = OUT.
  • Pilot orb. Bottom-right blue Earth thumbnail labelled GLOBE, visible when zoomed below 12 000 km. Tap = warp to global view.
  • Login orb. Pulsing aegean ◈ top-right when no session. Tap = auth panel.
  • Brain visibility. Every AI reply ends with via {provider} · {model} · {latency}s. Model accountability is not optional.
  • Brain is ours. Every prompt goes through aicycle. On failure, the chat shows the real error + a visible Retry button. Never silent fallback to another brain without architect permission.

Decision: how many minds to use

  • L0 (simple) → 1 mind
  • L1 (multi-part) → 2 minds
  • L2 (hard / strategic / "best" / "why") → 3 minds + judge

Routing one mind

  • news / today / price / weather → Perplexity (web-aware)
  • why / prove / debug / plan → DeepSeek (reasoning)
  • write / story / poem → OpenRouter (creative)
  • otherwise → Groq (fast)
  • if caller is_owner → Anthropic Opus (deep)

Memory recall — pgvector + gemini-embedding-001 @ 768 dims. Two pools: founder memories (Notis Astranov's voice) and per-user memories. Cosine similarity threshold 0.55. Top 8 founder + top 6 user memories appended to system prompt.

13 · Database Schema (minimum)

-- pgvector for memory recall
CREATE EXTENSION IF NOT EXISTS vector;

-- Memory store — founder & user knowledge
CREATE TABLE ai_memory (
  id         uuid PRIMARY KEY DEFAULT gen_random_uuid(),
  profile_id uuid REFERENCES auth.users(id) ON DELETE CASCADE,
  content    text NOT NULL,
  is_private boolean DEFAULT false,
  source     text,
  embedding  vector(768),
  created_at timestamptz DEFAULT now()
);
CREATE INDEX ai_memory_embedding_idx
  ON ai_memory USING ivfflat (embedding vector_cosine_ops);

-- Vector recall RPC
CREATE OR REPLACE FUNCTION match_memories(
  query_embedding vector(768),
  match_count int DEFAULT 12,
  profile_ids uuid[] DEFAULT NULL
) RETURNS TABLE (
  id uuid, profile_id uuid, content text, similarity float, is_owner boolean
) LANGUAGE sql STABLE AS $$
  SELECT m.id, m.profile_id, m.content,
         1 - (m.embedding <=> query_embedding) AS similarity,
         COALESCE(p.is_owner, false) AS is_owner
  FROM ai_memory m
  LEFT JOIN profiles p ON p.id = m.profile_id
  WHERE profile_ids IS NULL OR m.profile_id = ANY(profile_ids)
  ORDER BY m.embedding <=> query_embedding
  LIMIT match_count;
$$;

-- Audit log
CREATE TABLE cic_logs (
  id          uuid PRIMARY KEY DEFAULT gen_random_uuid(),
  profile_id  uuid,
  query       text,
  response    text,
  provider    text,
  via         text,
  latency_ms  int,
  created_at  timestamptz DEFAULT now()
);

-- Profiles (the people side)
CREATE TABLE profiles (
  id            uuid PRIMARY KEY REFERENCES auth.users(id),
  display_name  text,
  username      text,
  is_owner      boolean DEFAULT false,
  is_agent      boolean DEFAULT false,
  is_bot        boolean DEFAULT false,
  map_visibility text DEFAULT 'public',
  home_location geography(Point, 4326),
  eur_balance   numeric DEFAULT 0,
  avc_balance   numeric DEFAULT 0,
  bio           text,
  created_at    timestamptz DEFAULT now()
);

type Msg = { role: string; content: string }

async function embedText(geminiKey: string, text: string): Promise<number[] | null> {
  try {
    const model = 'models/gemini-embedding-001'
    const r = await fetch(`https://generativelanguage.googleapis.com/v1beta/${model}:embedContent?key=${geminiKey}`,
      { method: 'POST', headers: { 'content-type': 'application/json' },
        body: JSON.stringify({ model, content: { parts: [{ text: text.slice(0, 8000) }] }, outputDimensionality: 768 }) })
    if (!r.ok) return null
    const j = await r.json(); return Array.isArray(j.embedding?.values) ? j.embedding.values : null
  } catch { return null }
}

async function callAnthropic(key: string, model: string, system: string, messages: Msg[]) {
  try {
    const r = await fetch('https://api.anthropic.com/v1/messages', {
      method: 'POST',
      headers: { 'x-api-key': key, 'anthropic-version': '2023-06-01', 'content-type': 'application/json' },
      body: JSON.stringify({ model, max_tokens: 900, system, messages }),
    })
    if (!r.ok) return null
    const j = await r.json(); return j.content?.[0]?.text || null
  } catch { return null }
}

async function callOpenAICompat(url: string, key: string, model: string, system: string, messages: Msg[], extraHeaders: Record<string,string> = {}) {
  try {
    const r = await fetch(url, {
      method: 'POST',
      headers: { 'Authorization': `Bearer ${key}`, 'content-type': 'application/json', ...extraHeaders },
      body: JSON.stringify({ model, max_tokens: 900, messages: [{ role: 'system', content: system }, ...messages] }),
    })
    if (!r.ok) return null
    const j = await r.json(); return j.choices?.[0]?.message?.content || null
  } catch { return null }
}

async function callGemini(key: string, model: string, system: string, messages: Msg[]) {
  try {
    const contents = [
      { role: 'user',  parts: [{ text: system }] },
      { role: 'model', parts: [{ text: 'Understood. I am Astranov.' }] },

    const j = await r.json(); return j.candidates?.[0]?.content?.parts?.[0]?.text || null
  } catch { return null }

    const connected = ALL.filter(k => E(ENV_OF[k]))

    let raw: string | null = null, engine = '', engineModel = '', participants: string[] = [], routedBy = ''

    function routeEngine(): string {
      const p = prompt.toLowerCase(); const has = (k: string) => connected.includes(k)
      if (/(today|latest|news|current|price|weather|score|recent|happening|stock)/.test(p) && has('perplexity')) { routedBy = 'fresh/web'; return 'perplexity' }
      if (/(why|prove|calculate|debug|algorithm|reason|analy|architecture|plan|strategy|compare|math)/.test(p) && has('deepseek')) { routedBy = 'reasoning'; return 'deepseek' }
      if (/(write|imagine|story|poem|creative|design|brainstorm|slogan)/.test(p) && has('openrouter')) { routedBy = 'creative'; return 'openrouter' }
      if (has('groq')) { routedBy = 'fast'; return 'groq' }
      routedBy = 'default'; return connected[0] || 'openrouter'
    }
    function problemLevel(): number {
      const lw = prompt.toLowerCase()
      const hard = /(best|compare|analy|deep|strateg|decide|recommend|pros and cons|\bwhy\b|prove|architecture|invest|legal|medical|complex|in detail|step by step|trade-?off|evaluate)/.test(lw)
      if (hard || prompt.length > 240) return 2
      if ((prompt.match(/\?/g) || []).length > 1 || prompt.length > 120) return 1
      return 0
    }

    async function judge(answers: { key: string; raw: string }[]): Promise<string | null> {
      const jSys = system + `\n\nYou are the AstranoV judgment. Several of your minds answered below. Read them, then give the SINGLE best answer in your own first-person voice — merge what is true and useful, drop the weak, stay concise. NEVER mention multiple minds or name any of them.`
      const jUser = `The user said: ${prompt}\n\nYour minds answered:\n` + answers.map((a, i) => `[#${i+1}]\n${a.raw}`).join('\n\n') + `\n\nGive the final Astranov answer now.`
      const jm: Msg[] = [{ role: 'user', content: jUser }]
      if (isOwner && ANTHROPIC) { const r = await callAnthropic(ANTHROPIC, ANTHRO_MODEL, jSys, jm); if (r) return r }
      const orK = E('OPENROUTER_API_KEY'); if (orK) { const r = await callOpenAICompat('https://openrouter.ai/api/v1/chat/completions', orK, E('OPENROUTER_MODEL') || 'meta-llama/llama-3.3-70b-instruct', jSys, jm, { 'HTTP-Referer': 'https://astranov.eu', 'X-Title': 'AstranoV' }); if (r) return r }
      const gqK = E('GROQ_API_KEY');       if (gqK) { const r = await callOpenAICompat('https://api.groq.com/openai/v1/chat/completions', gqK, E('GROQ_MODEL') || 'llama-3.3-70b-versatile', jSys, jm); if (r) return r }
      return null
    }

    async function runMany(keys: string[], withOwner: boolean) {
      const runs = keys.map(async k => { const o = await RUN[k](); return o && o.raw ? { key: k, raw: o.raw, model: o.model } : null })
      if (withOwner && isOwner && ANTHROPIC) runs.push((async () => { const r = await callAnthropic(ANTHROPIC, ANTHRO_MODEL, system, messages); return r ? { key: 'anthropic', raw: r, model: ANTHRO_MODEL } : null })())
      return (await Promise.all(runs)).filter((x): x is { key: string; raw: string; model: string } => !!x)
    }

    const isLock = enginePref && !['astranov','auto','all'].includes(enginePref) && !!RUN[enginePref]
    if (isLock) {
      const o = await RUN[enginePref](); if (o && o.raw) { raw = o.raw; engine = enginePref; engineModel = o.model }
      if (!raw) for (const k of shuffled) { const o2 = await RUN[k](); if (o2 && o2.raw) { raw = o2.raw; engine = k; engineModel = o2.model; break } }
      participants = engine ? [engine] : []; routedBy = 'locked'
    } else {
      const lvl = escalate ? 2 : problemLevel()
      const count = lvl === 0 ? 1 : (lvl === 1 ? 2 : 3)
      if (count === 1) {
        const routed = routeEngine()
        if (isOwner && ANTHROPIC && (routedBy === 'default' || routedBy === 'fast')) { raw = await callAnthropic(ANTHROPIC, ANTHRO_MODEL, system, messages); if (raw) { engine = 'anthropic'; engineModel = ANTHRO_MODEL; routedBy = 'owner-deep' } }
        if (!raw && RUN[routed]) { const o = await RUN[routed](); if (o && o.raw) { raw = o.raw; engine = routed; engineModel = o.model } }
        if (!raw) for (const k of shuffled) { const o2 = await RUN[k](); if (o2 && o2.raw) { raw = o2.raw; engine = k; engineModel = o2.model; routedBy = 'fallback'; break } }
        participants = engine ? [engine] : []
      } else {
        const settled = await runMany(shuffled.slice(0, count), lvl === 2)
        participants = settled.map(s => s.key)
        routedBy = escalate ? 'escalated' : ('auto-L' + lvl)
        if (settled.length === 1)      { raw = settled[0].raw; engine = settled[0].key; engineModel = settled[0].model }
        else if (settled.length > 1)   { raw = (await judge(settled)) || settled[0].raw; engine = 'astranov'; engineModel = 'ensemble:' + participants.join('+') }
        else                           { const routed = routeEngine(); const o = RUN[routed] ? await RUN[routed]() : null; if (o && o.raw) { raw = o.raw; engine = routed; engineModel = o.model; participants = [routed] } }
      }
    }

    if (!raw) return json({ response: 'Astranov is gathering itself — try again in a moment.', text: 'Astranov is gathering itself — try again in a moment.', provider: 'astranov', engineLabel: 'Astranov', participants, routedBy })

    // Teach intent — "remember / θυμήσου / note that" persists to ai_memory
    try {
      const lower = prompt.toLowerCase()
      const isTeach = /^\s*(remember|note that|keep in mind|don'?t forget|θυμήσου|να θυμάσαι)\b/.test(lower)
      if (isTeach && profileId && prompt.length >= 10) {
        const content = prompt.replace(/^\s*(remember|note that|keep in mind|don'?t forget|θυμήσου|να θυμάσαι)[:,]?\s*/i, '').slice(0, 1000)
        if (content.length >= 4) {
          const emb = GEMINI ? await embedText(GEMINI, content) : null
          await supabase.from('ai_memory').insert({ profile_id: profileId, content, is_private: false, source: isOwner ? 'creator-taught' : 'user-taught', embedding: emb })
        }
      }
    } catch (e) { console.error('memory learn:', e) }

    const latencyMs = Date.now() - t0
    try { await supabase.from('cic_logs').insert({ profile_id: profileId, query: prompt.slice(0, 2000), response: raw.slice(0, 4000), provider: engine || 'astranov', via: routedBy, latency_ms: latencyMs }) } catch {}

    return json({
      response: raw, text: raw, provider: 'astranov', label: 'Astranov',
      engine, engineLabel: ENGINE_LABELS[engine] || 'Astranov', model: engineModel,
      latencyMs, mode: mode || 'adaptive', escalated: escalate,
      participants, routedBy,
      recalled: { creator: creatorMind.length, user: userMemory.length },
    })
  } catch (e) {
    console.error('aicycle error:', e)
    return json({ response: 'Something went wrong.', text: 'Something went wrong.', provider: 'error', engineLabel: 'Astranov' }, 500)
  }
})
